# Lung-Cancer-Risk-Predictors-ML
A machine learning project predicting lung cancer risk based on lifestyle factors. Compares SVM, KNN, and ANN models while uncovering unconventional correlations between digital habits (Social Media, Gaming) and health outcomes. | 基於生活習慣的肺癌風險預測專案。透過比較 SVM、KNN 與 ANN 模型，並揭示社群媒體與線上遊戲等現代習慣與肺癌風險之間的非典型關聯。
